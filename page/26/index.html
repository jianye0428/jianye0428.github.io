PHP 表单安全性的重要提示
$_SERVER["PHP_SELF"]变量能够被黑客利用！
如果页面中使用了PHP_SELF，用户能够输入下划线然后执行跨站点脚本（XSS）。
比如说注入 js 脚本等，valine 以前的版本也有过这样的漏洞。

跨站点脚本（Cross-site scripting，XSS）是一种计算机安全漏洞类型，常见于 Web 应用程序。XSS 能够使攻击者向其他用户浏览的网页中输入客户端脚本。
